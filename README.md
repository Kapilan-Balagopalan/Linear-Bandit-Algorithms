Linear Bandits

The repository provides code for below state-of-the-art Linear bandits algorithms and evaluates their performance for various problem instances. We have implemented multiple processes version as well to save computation time.


Implemented algorithms

1. LinMED (ours)
2. LinMEDNOPT (ours)
3. OFUL
4. LinIMED – 1,2,3
5. LinTS – Frequentest
6. LinTS – Bayesian
7. EXP2
8. SpannerIGW

Problem Instances

1. End of optimism
2. Large gap instance
3. Synthetic unit ball arm set
4. Real world data set – Movielens
5. Delayed reward setting
6. Offline evaluation

Configurations

1. Single process
2. Multiple processes
